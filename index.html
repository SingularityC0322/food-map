初次配置

需要连接你的 Supabase 数据库，只需设置一次，之后所有设备都能同步数据。

步骤 1

注册 Supabase

前往 supabase.com 注册免费账号，创建一个新项目（Project）。名称随意，记住数据库密码。

步骤 2

创建数据表

进入你的项目 → 左侧「SQL Editor」→ 粘贴以下 SQL 并点击 Run：

create table entries ( id bigint generated always as identity primary key, name text not null, type text not null, cuisine_id text not null, cuisine_name text not null, place text, note text, rating text, want_again boolean default false, date text, created_at timestamptz default now() ); alter table entries enable row level security; create policy "allow all" on entries for all using (true) with check (true);
